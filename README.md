\# Sales Dashboard 2020-22 – Power BI



\## Overview

Interactive sales dashboard built in Power BI Desktop to analyze 

2020-22 sales data across products, brands, states, and supervisors.



\## Key Metrics

\- Total Sales: 18M

\- Total Cost: 14M

\- Total Profit: 4M

\- Total Transactions: 948

\- Total Quantity: 2K



\## Features

\- Supervisor-wise filtering using interactive slicer

\- Quantity by Category bar chart (Monitor, Mouse, CPU, etc.)

\- State-wise sales using Bing Map visualization

\- Brand-wise sales breakdown via Pie Chart (Intel, Samsung, Dell, etc.)

\- KPI cards for quick business overview



\## Tools Used

Power BI Desktop | DAX | Data Modeling
